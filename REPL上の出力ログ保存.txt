● パイプラインの構築が完了しました！

  完成した内容

  以下のファイルを作成しました：

  1. ocr.py - OCRモジュール

  - PDFファイルを画像に変換（PyMuPDFを使用）
  - 各ページからOCRでテキストを抽出（Tesseract + pytesseractを使用）
  - UTF-8エンコーディングでテキストファイルを保存

  2. normalize.py - 正規化モジュール

  - OCRで抽出したテキストを解析
  - 班番号、氏名、住所、TEL、メールアドレスを抽出
  - 希望部会を個別の部会カラムに展開（①、②、③を対応する部会に配置）
  - 補足事項を抽出
  - 全角数字を半角数字に変換

  3. to_excel.py - Excel/CSV出力モジュール

  - 15個のカラムでExcelファイルを作成
  - CSVファイルも同時に作成（UTF-8エンコーディング）

4. run.py - パイプライン実行スクリプト

  - 全体のパイプラインを実行
  - ログ出力とエラーハンドリング

  5. requirements.txt - 依存パッケージ

  出力結果

  output2/ディレクトリに以下のファイルが生成されました：

  1. 班長からの役員希望内訳_extracted.txt - OCR抽出テキスト
  2. 班長からの役員希望内訳_output.xlsx - Excelファイル（15カラム）
  3. 班長からの役員希望内訳_output.csv - CSVファイル（UTF-8）

  23件の班長エントリーを解析し、希望部会が各部会カラム（事務局、会計、書記、名簿、防犯防災、回覧広報、地域コミュ、環境美化、厚生福祉）に①②③として正しく配置されています。

  使用方法

  # 基本的な使い方
  python run.py imgFile/班長からの役員希望内訳.pdf

  # 出力ディレクトリを指定
  python run.py imgFile/班長からの役員希望内訳.pdf --output-dir 出力先フォルダ

  # 韓国語でOCR処理
  python run.py input.pdf --lang kor

  # ヘルプを表示
  python run.py --help

  出力ファイルは指定したディレクトリ（デフォルト：output）に保存されます。
